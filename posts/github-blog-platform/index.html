VSC Blog IDE

  1. 安装Visual Studio Code，也可在Windows应用商店中安装；
  2. 安装用于VSC的git
  3. 安装Windows桌面版git
  4. 安装git 三个git均可同时安装，可在VSC内提交、也可在桌面版的git内提交
  5. 使用桌面版git克隆blog；
  6. 在VSC内将blog目录qin-dong.github.io设置为工作区主目录；
  7. 安装VSC扩展PasteImage，并进行相应设置
  8. 安装VSC扩展MarkDown All in One。

  • VSC内粘贴快捷键：Ctrl+Alt+V
